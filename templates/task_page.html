Task description! In CSS, you can query based on the selector for radios and checkboxes to style siblings down the DOM scope. To do this, we can use the. It will select same-level siblings after the given selector. Because the tab in this demo are nested and not immediate siblings, we will need to select their topmost parent that is at the same level as our .

To demonstrate, we will do a simplified version of this with a checkbox:

Example 1

<input id="demo-child-toggle" type="checkbox">
<label for="demo-child-toggle">Toggle #demo-child</label>
<div id="demo-parent">
<div id="demo-child">#demo-child</div>
</div>

Example 2

#demo-child-toggle {
  display: none;
}
#demo-child-toggle:checked ~ #demo-parent #demo-child {
  color: #c0392b;
  font-weight: bold;
  text-transform: uppercase;
}
#note

As you can see, we can control the style of content that comes after a hidden input by toggling it via its label.
